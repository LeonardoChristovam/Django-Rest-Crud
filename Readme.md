### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/LeonardoChristovam/Django-Rest-Crud.git
   ```
2. Activate Enviroment
   ```sh
   .\env\Scripts\activate
   ```
3. Install Django Rest Framework
   ```sh
   pip install django djangorestframework
   ```
